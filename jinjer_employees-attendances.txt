従業員 打刻データ

従業員に紐づく打刻データ情報を取り扱います。
/v2/employees/attendances

従業員に紐づく打刻データを取得する。
Authorizations:
ApplicationJWT
query Parameters
month
required
	
string <date> yyyy-MM
Example: month=2023-04

取得する打刻データの年月を指定します。
年月が指定されていない場合、エラーを返却します。
page	
integer
Example: page=1

クエリパラメータが無い場合、20件返却します。
21-40件目を取得したい場合、page=2 を指定してください。
employee-ids	
string
Example: employee-ids=1,2,3...,100

複数の社員番号をカンマ区切りで指定できます。
20件以上含まれる場合は、20件までを返します。
クエリパラメータが無い場合、条件に含めずデータを返却します。
