従業員 給与計算結果

従業員に紐づく給与計算結果を取り扱います。
/v1/employees/salary-statements

従業員に紐づく給与計算結果を取得する。
Authorizations:
ApplicationJWT
HTTP: ApplicationJWT
HTTP Authorization Scheme: bearer
Bearer format: JWT
query Parameters
executed-on
required
	
string <date> yyyy-MM
Example: executed-on=2024-01

指定された処理月のデータを返却します。
処理月が指定されていない場合、エラーを返却します。
page	
integer
Example: page=1

クエリパラメータが無い場合、10件返却します。
11-20件目を取得したい場合、page=2 を指定してください。
employee-ids	
string
Example: employee-ids=1,2,3...,100

複数の社員番号をカンマ区切りで指定できます。
100件以上含まれる場合は、100件までを返します。
クエリパラメータが無い場合、条件に含めずデータを返却します。
has-since-changed-at	
string <date> yyyy-MM-dd
Example: has-since-changed-at=2023-01-01

指定された年月日以降に新規登録または更新されたデータを返却します。
年月日が指定されていなかったら全データを返却します。
